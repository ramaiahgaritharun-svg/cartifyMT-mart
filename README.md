# CartifyMT-Mart 🛒

CartifyMT-Mart is a full-stack e-commerce application that I built to strengthen my skills in modern web development using React and Django REST Framework. The project simulates a real-world online shopping platform where users can browse products, search and filter items, manage their cart, and place orders, while administrators can manage products, categories, and platform data through a dedicated dashboard.

## Why I Built This Project

As part of my journey toward becoming a Full Stack Developer, I wanted to build a project that combines frontend development, backend APIs, authentication, database management, and role-based access control. CartifyMT-Mart helped me gain practical experience in connecting a React frontend with a Django REST backend and implementing features commonly found in production e-commerce platforms.

## Features

### Customer Features

* User Registration and Login
* JWT-based Authentication
* Browse Products
* Product Search and Category Filtering
* Product Details View
* Add Products to Cart
* Order Placement and Tracking

### Admin Features

* Admin Dashboard
* Add, Update, and Delete Products
* Manage Categories
* Monitor Orders
* Manage Platform Data
